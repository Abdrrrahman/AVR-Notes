# Linked List Overview
A linked list is a linear data structure that stores elements in a sequence, just like an array. However, unlike arrays, linked list elements are **not stored contiguously in memory**. Instead, every element (called a **node**) stores two pieces of information:

- The actual data.
- The address of the next node in the sequence.

This address acts as a link between consecutive nodes, which is where the name **Linked List** comes from.

Unlike arrays, linked lists do not have a fixed size. Every new node is allocated dynamically on the heap using `malloc()`, allowing the list to grow until the system runs out of available memory.

The first node in the linked list is called the **head**, and a pointer named `head` always stores the address of the first node. If the linked list is empty, the head pointer stores `nullptr` (`NULL` in standard C).

Unlike arrays, linked lists allow insertion and deletion without shifting elements. However, accessing an element by index requires traversing the list from the beginning because linked lists do not support direct indexing.

The implementation we will discuss here is a **Singly Linked List**, meaning every node stores the address of only the next node.

Unlike arrays, linked lists do not store their elements consecutively in memory. Therefore, we first need to define what a single node looks like before defining the list itself.

Each node stores three pieces of information:

- A generic pointer to the stored data.
- The datatype of the stored data.
- A pointer to the next node.

### Node struct

```c
// Code Snippet 1-2

typedef struct Node {
	void *data;
	Datatype type;
	struct Node *next;
} Node;
```

The first member, `data`, stores the address of the actual data rather than the data itself. This makes our linked list generic, allowing it to store any datatype.

The second member, `type`, stores an enum value representing the datatype of `data`. This allows us to display the stored data correctly using `genericPrint()`.

The final member, `next`, stores the address of the following node in the linked list. If the current node is the last node, then `next` stores `nullptr`.

### LinkedList struct

```c
// Code Snippet 1-3

typedef struct {
	Node *head;
	int size;
} LinkedList;
```

After defining a node, we define the linked list itself.

Unlike the stack, the linked list does not have a stack pointer or top pointer. Instead, it stores a pointer named `head`, which always points to the first node in the list.

We also store an integer named `size`, which keeps track of the current number of nodes in the linked list. This saves us from traversing the entire list every time we want to know its length.

Finally, we declare the prototypes of all methods used to manipulate the linked list. Every method will be explained individually in the implementation section below.

## LinkedList.c

```c
// Code Snippet 1-4

#include <stdio.h>
#include <stdlib.h>

#include "LinkedList.h"

void initializeList(LinkedList *list) {
	list->head = nullptr;
	list->size = 0;
}

void insertAtBeginning(LinkedList *list, void *data, Datatype type) {
	Node *newNode = malloc(sizeof(Node));
	
	if(newNode == nullptr) {
		fprintf(stderr, "Memory Allocation Failed!\n");
	} else {
		newNode->data = data;
		newNode->type = type;
		newNode->next = list->head;
		
		list->head = newNode;
		list->size++;
	}
}

void insertAtEnd(LinkedList *list, void *data, Datatype type) {
	Node *newNode = malloc(sizeof(Node));
	
	if(newNode == nullptr) {
		fprintf(stderr, "Memory Allocation Failed!\n");
		return;
	}
	
	newNode->data = data;
	newNode->type = type;
	newNode->next = nullptr;
	
	if(isEmpty(list)) {
		list->head = newNode;
	} else {
		Node *current = list->head;
		
		while(current->next != nullptr) {
			current = current->next;
		}
		
		current->next = newNode;
	}	
	list->size++;
}

void insertAtIndex(LinkedList *list, int index, void *data, Datatype type) {
	if(index < 0 || index > list->size) {
		fprintf(stderr, "Index out of range!\n");
		return;
	}
	
	if(index == 0) {
		insertAtBeginning(list, data, type);
		return;
	}
	
	if(index == list->size) {
		insertAtEnd(list, data, type);
		return;
	}
	
	Node *newNode = malloc(sizeof(Node));
	
	if(newNode == nullptr) {
		fprintf(stderr, "Memory Allocation Failed!\n");
		return;
	}
	
	newNode->data = data;
	newNode->type = type;
	
	Node *current = list->head;
	
	for(int i = 0; i < index - 1; i++) {
		current = current->next;
	}
	
	newNode->next = current->next;
	current->next = newNode;
	
	list->size++;
}

Node *search(const LinkedList *list, void *data) {
	Node *current = list->head;
	
	while(current != nullptr) {
		if(current->data == data) {
			return current;
		}
		
		current = current->next;
	}
	
	return nullptr;
}

void removeByIndex(LinkedList *list, int index) {
	if(index < 0 || index >= list->size) {
		fprintf(stderr, "Index out of range!\n");
		return;
	}
	
	Node *deleteMe;
	
	if(index == 0) {
		deleteMe = list->head;
		list->head = list->head->next;
	} else {
		Node *current = list->head;
		
		for(int i = 0; i < index - 1; i++) {
			current = current->next;
		}
		
		deleteMe = current->next;
		current->next = deleteMe->next;
	}
	
	free(deleteMe);
	list->size--;
}

void removeByValue(LinkedList *list, void *data) {
	if(isEmpty(list)) {
		fprintf(stderr, "Linked list is empty!\n");
		return;
	}
	
	if(list->head->data == data) {
		removeByIndex(list, 0);
		return;
	}
	
	Node *current = list->head;
	
	while(current->next != nullptr &&
	      current->next->data != data) {
		current = current->next;
	}
	
	if(current->next == nullptr) {
		fprintf(stderr, "Data was not found!\n");
		return;
	}
	
	Node *deleteMe = current->next;
	current->next = deleteMe->next;
	
	free(deleteMe);
	list->size--;
}

int isEmpty(const LinkedList *list) {
	return (list->head == nullptr);
}

int numberOfElements(const LinkedList *list) {
	return list->size;
}

void displayElements(const LinkedList *list) {
	if(isEmpty(list)) {
		fprintf(stderr, "Can not display an empty linked list!\n");
	} else {
		Node *current = list->head;
		
		while(current != nullptr) {
			genericPrint(current->data, current->type);
			current = current->next;
		}
	}
}

void destroyList(LinkedList *list) {
	while(!isEmpty(list)) {
		removeByIndex(list, 0);
	}
}
```

The linked list implementation relies on dynamic memory allocation; therefore, we include `<stdlib.h>` to use `malloc()` and `free()`. We also include `<stdio.h>` for printing error messages.

### `void initializeList(LinkedList *list);`

This method initializes the linked list by assigning the default values to its members. Since every linked list object has its own head pointer and size variable, we must initialize every newly created list before using it.

An empty linked list has no first node, so `head` points to `nullptr`. Likewise, since there are no nodes yet, `size` is initialized to `0`.

```c
// Code Snippet 1-5

void initializeList(LinkedList *list) {
	list->head = nullptr;
	list->size = 0;
}
```

### `void insertAtBeginning(LinkedList *list, void *data, Datatype type);`

This method inserts a new node at the beginning of the linked list.

First, it allocates memory for a new node using `malloc()`. Since dynamic memory allocation may fail, it checks whether the returned pointer is `nullptr`. If allocation succeeds, it initializes the node by storing the entered data and datatype.

Next, the `next` pointer of the newly created node is assigned to the current head of the list. This makes the new node point to what was previously the first node.

Finally, the head pointer is updated to point to the new node, making it the first node in the linked list. Then, the size is incremented by one.

```c
// Code Snippet 1-6

void insertAtBeginning(LinkedList *list, void *data, Datatype type) {
	Node *newNode = malloc(sizeof(Node));
	
	if(newNode == nullptr) {
		fprintf(stderr, "Memory Allocation Failed!\n");
	} else {
		newNode->data = data;
		newNode->type = type;
		newNode->next = list->head;
		
		list->head = newNode;
		list->size++;
	}
}
```

### `void insertAtEnd(LinkedList *list, void *data, Datatype type);`

This method inserts a new node at the end of the linked list.

First, it allocates memory for a new node and initializes its members. Since the new node will become the last node in the list, its `next` pointer is initialized to `nullptr`.

If the linked list is empty, the newly created node simply becomes the head of the list.

Otherwise, the method traverses the linked list beginning from the head until it reaches the last node. The last node is identified by having its `next` pointer equal to `nullptr`.

Once the last node is reached, its `next` pointer is assigned to the newly created node, effectively attaching the node to the end of the linked list.

Finally, the list size is incremented.

```c
// Code Snippet 1-7

void insertAtEnd(LinkedList *list, void *data, Datatype type) {
	Node *newNode = malloc(sizeof(Node));
	
	if(newNode == nullptr) {
		fprintf(stderr, "Memory Allocation Failed!\n");
		return;
	}
	
	newNode->data = data;
	newNode->type = type;
	newNode->next = nullptr;
	
	if(isEmpty(list)) {
		list->head = newNode;
	} else {
		Node *current = list->head;
		
		while(current->next != nullptr) {
			current = current->next;
		}
		
		current->next = newNode;
	}
	
	list->size++;
}
```

### `void insertAtIndex(LinkedList *list, int index, void *data, Datatype type);`

This method inserts a new node at a specific index in the linked list.

First, it validates the entered index. A valid insertion index ranges from **0** to **size**, since inserting at `size` simply means inserting at the end of the list.

If the index is `0`, the insertion becomes identical to inserting at the beginning, so the method simply calls `insertAtBeginning()`.

Likewise, if the index is equal to the size of the linked list, the insertion becomes identical to inserting at the end, so `insertAtEnd()` is called.

Otherwise, a new node is allocated and initialized. Then the method traverses the linked list until it reaches the node immediately before the desired insertion position.

Finally, the links are updated in two steps. First, the new node points to the node currently occupying the desired position. Then the previous node is updated to point to the new node. This inserts the new node between the two existing nodes without losing any part of the linked list.

```c
// Code Snippet 1-8

void insertAtIndex(LinkedList *list, int index, void *data, Datatype type) {
	if(index < 0 || index > list->size) {
		fprintf(stderr, "Index out of range!\n");
		return;
	}
	
	if(index == 0) {
		insertAtBeginning(list, data, type);
		return;
	}
	
	if(index == list->size) {
		insertAtEnd(list, data, type);
		return;
	}
	
	Node *newNode = malloc(sizeof(Node));
	
	if(newNode == nullptr) {
		fprintf(stderr, "Memory Allocation Failed!\n");
		return;
	}
	
	newNode->data = data;
	newNode->type = type;
	
	Node *current = list->head;
	
	for(int i = 0; i < index - 1; i++) {
		current = current->next;
	}
	
	newNode->next = current->next;
	current->next = newNode;
	
	list->size++;
}
```

```bash
Before:

current ---> [ A ] ----> [ C ]

After inserting B:

current ---> [ A ] ----> [ B ] ----> [ C ]
```

> **Note:** Although one of the MISRA C rules tells us to use only one return per function and place it at the end, it is an Advisory rule that is applied when the function's complexity is weak. If the function is complex or large, it is not preferred to use this rule in order to preserve ease of readability.

### `Node *search(const LinkedList *list, void *data);`

This method searches the linked list for a specific piece of data and returns the address of the node containing it.

The method begins by creating a pointer named `current` that starts at the head of the linked list.

It then traverses the linked list one node at a time until either the desired node is found or the end of the linked list is reached.

If the data stored inside the current node matches the entered data, the method immediately returns the address of that node.

If no matching node is found after traversing the entire linked list, the method returns `nullptr`.

Notice that this implementation compares the stored pointers themselves. Therefore, two different pointers that happen to store equal values are considered different.

```c
// Code Snippet 1-9

Node *search(const LinkedList *list, void *data) {
	Node *current = list->head;
	
	while(current != nullptr) {
		if(current->data == data) {
			return current;
		}
		
		current = current->next;
	}
	
	return nullptr;
}
```

### `void removeByIndex(LinkedList *list, int index);`

This method removes the node located at a specific index in the linked list.

The method first validates the entered index. Since removal requires an existing node, the valid range is from **0** to **size - 1**.

If the index is zero, then the first node (head) is removed. A temporary pointer named `deleteMe` stores the current head node. Then the head pointer advances to the second node in the linked list. Finally, the old head node is deallocated.

If the node is not the head, the linked list is traversed until reaching the node immediately before the node to be removed.

Once this node is reached, another temporary pointer (`deleteMe`) stores the address of the node to be deleted. The previous node is then linked directly to the following node by assigning its `next` pointer to `deleteMe->next`.

After reconnecting the linked list, the removed node is deallocated using `free()`, and the size of the linked list is decremented.

```c
// Code Snippet 1-10

void removeByIndex(LinkedList *list, int index) {
	if(index < 0 || index >= list->size) {
		fprintf(stderr, "Index out of range!\n");
		return;
	}
	
	Node *deleteMe;
	
	if(index == 0) {
		deleteMe = list->head;
		list->head = list->head->next;
	} else {
		Node *current = list->head;
		
		for(int i = 0; i < index - 1; i++) {
			current = current->next;
		}
		
		deleteMe = current->next;
		current->next = deleteMe->next;
	}
	
	free(deleteMe);
	list->size--;
}
```

```bash
After removing B:
```

### `void removeByValue(LinkedList *list, void *data);`

This method removes the first node containing the specified data.

It first checks whether the linked list is empty, as removing from an empty linked list is not possible.

Next, it checks whether the head node contains the desired data. If so, removing the node is identical to removing the node at index zero, so the method simply calls `removeByIndex()`.

Otherwise, the linked list is traversed until reaching the node immediately before the node containing the desired data.

If the traversal reaches the end of the linked list without finding the data, an error message is displayed and the method returns.

If the node is found, a temporary pointer stores the node that will be removed. The previous node is then linked directly to the following node, bypassing the removed node.

Finally, the removed node is deallocated and the size of the linked list is decremented.

```c
// Code Snippet 1-11

void removeByValue(LinkedList *list, void *data) {
	if(isEmpty(list)) {
		fprintf(stderr, "Linked list is empty!\n");
		return;
	}
	
	if(list->head->data == data) {
		removeByIndex(list, 0);
		return;
	}
	
	Node *current = list->head;
	
	while(current->next != nullptr &&
	      current->next->data != data) {
		current = current->next;
	}
	
	if(current->next == nullptr) {
		fprintf(stderr, "Data was not found!\n");
		return;
	}
	
	Node *deleteMe = current->next;
	current->next = deleteMe->next;
	
	free(deleteMe);
	list->size--;
}
```

### `int isEmpty(const LinkedList *list);`

This method is a boolean method that returns whether the linked list is empty or not. Since C does not have a built-in `bool` datatype, we use `int`, where `1` represents **true** and `0` represents **false**.

A linked list is considered empty when its head pointer points to `nullptr`, meaning there is no first node in the list.

```c
// Code Snippet 1-12

int isEmpty(const LinkedList *list) {
	return (list->head == nullptr);
}
```

### `int numberOfElements(const LinkedList *list);`

This method returns the number of nodes currently stored in the linked list.

Since the linked list maintains a member named `size`, there is no need to traverse the linked list every time we want to know its length. The value is simply returned directly.

```c
// Code Snippet 1-13

int numberOfElements(const LinkedList *list) {
	return list->size;
}
```

### `void displayElements(const LinkedList *list);`

This method displays every element stored in the linked list from the head node to the last node.

First, it checks whether the linked list is empty. If so, an error message is displayed.

Otherwise, a temporary pointer named `current` is initialized to point to the head of the linked list. The method then traverses the linked list one node at a time until it reaches the end of the list.

During each iteration, the node's data is printed using `genericPrint()`. Then `current` advances to the following node.

The traversal terminates when `current` becomes `nullptr`, indicating that there are no more nodes to visit.

```bash
# Bash Snippet 1-3

head
 │
 ▼
+------+------+      +------+------+      +------+------+
| Data | Next | ───► | Data | Next | ───► | Data | NULL |
+------+------+      +------+------+      +------+------+
     ▲
     │
 current
```

At every iteration, `current` moves one node forward until it reaches `nullptr`.

```c
// Code Snippet 1-14

void displayElements(const LinkedList *list) {
	if(isEmpty(list)) {
		fprintf(stderr, "Can not display an empty linked list!\n");
	} else {
		Node *current = list->head;
		
		while(current != nullptr) {
			genericPrint(current->data, current->type);
			current = current->next;
		}
	}
}
```

### `void destroyList(LinkedList *list);`

This method deallocates every node in the linked list. It serves a role similar to a destructor in Object-Oriented Programming, ensuring that all dynamically allocated memory is released before the linked list is no longer needed.

Since every node in the linked list was allocated using `malloc()`, each one must eventually be deallocated using `free()`. Failing to do so results in **memory leaks**, where allocated memory remains occupied even though it is no longer accessible.

Rather than manually traversing the linked list and freeing every node, this method repeatedly removes the first node by calling `removeByIndex(list, 0)`. Each call removes the current head node, advances the head pointer to the following node, and deallocates the removed node.

The loop continues until the linked list becomes empty.

```bash
# Bash Snippet 1-4

Before:

head
 │
 ▼
[A] ───► [B] ───► [C] ───► NULL

removeByIndex(list, 0)

head
 │
 ▼
[B] ───► [C] ───► NULL

removeByIndex(list, 0)

head
 │
 ▼
[C] ───► NULL

removeByIndex(list, 0)

head
 │
 ▼
NULL
```

At this point, every allocated node has been deallocated, leaving an empty linked list.

```c
// Code Snippet 1-15

void destroyList(LinkedList *list) {
	while(!isEmpty(list)) {
		removeByIndex(list, 0);
	}
}
```
