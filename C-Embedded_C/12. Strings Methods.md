These functions are used to manipulate strings in C programming language, All of them are found in `string.h` libraray.
# String Copying Functions

### `strcpy`
```c
char *strcpy(char *dest, const char *src);
```
**Purpose:** Copies the string pointed to by `src` to the buffer pointed to by `dest`.
**Parameters:**
- `dest` - Destination buffer (must be large enough)
- `src` - Source string to copy
**Returns:** Pointer to `dest`
**Notes:** Does not check buffer bounds. Copies until null terminator is found.
**Example:**
```c
char dest[20];
strcpy(dest, "Hello World");  // dest now contains "Hello World"
```
---
### `strncpy`
```c
char *strncpy(char *dest, const char *src, size_t n);
```
**Purpose:** Copies up to `n` characters from `src` to `dest`.
**Parameters:**
- `dest` - Destination buffer
- `src` - Source string
- `n` - Maximum number of characters to copy
**Returns:** Pointer to `dest`
**Notes:**
- If `src` is shorter than `n`, pads with null bytes
- If `src` is longer than `n`, does NOT null-terminate `dest`
- Safer than `strcpy` but requires careful handling
**Example:**
```c
char dest[10];
strncpy(dest, "Hello World", 5);  // dest contains "Hello" (may not be null-terminated)
dest[5] = '\0';  // Ensure null termination
```
---
# String Concatenation Functions
### `strcat`
```c
char *strcat(char *dest, const char *src);
```
**Purpose:** Appends the string pointed to by `src` to the end of the string pointed to by `dest`.
**Parameters:**
- `dest` - Destination string (must have enough space)
- `src` - Source string to append
**Returns:** Pointer to `dest`
**Notes:** `dest` must be large enough to hold both strings plus null terminator.
**Example:**
```c
char dest[20] = "Hello ";
strcat(dest, "World");  // dest now contains "Hello World"
```
---
### `strncat`
```c
char *strncat(char *dest, const char *src, size_t n);
```
**Purpose:** Appends up to `n` characters from `src` to `dest`.
**Parameters:**
- `dest` - Destination string
- `src` - Source string to append
- `n` - Maximum number of characters to append
**Returns:** Pointer to `dest`
**Notes:** Always null-terminates the result.
**Example:**
```c
char dest[20] = "Hello ";
strncat(dest, "World!!!", 5);  // dest contains "Hello World"
```

# String Comparison Functions

### `strcmp`
```c
int strcmp(const char *str1, const char *str2);
```
**Purpose:** Compares two strings lexicographically.
**Parameters:**
- `str1` - First string
- `str2` - Second string
**Returns:**
- `0` if strings are equal
- Negative if `str1` < `str2`
- Positive if `str1` > `str2`
**Example:**
```c
int result = strcmp("apple", "banana");  // Returns negative value
```
---
### `strncmp`
```c
int strncmp(const char *str1, const char *str2, size_t n);
```
**Purpose:** Compares up to `n` characters of two strings.
**Parameters:**
- `str1` - First string
- `str2` - Second string
- `n` - Maximum number of characters to compare
**Returns:** Same as `strcmp` but only for first `n` characters
**Example:**
```c
int result = strncmp("hello", "help", 3);  // Returns 0 (first 3 chars equal)
```
---
### `strcoll`
```c
int strcoll(const char *str1, const char *str2);
```
**Purpose:** Compares two strings according to the current locale.
**Parameters:**
- `str1` - First string
- `str2` - Second string
**Returns:** Same as `strcmp` but locale-aware
**Notes:** Useful for internationalization where character ordering varies by locale.

# String Search Functions
### `strchr`
```c
char *strchr(const char *str, int c);
```
**Purpose:** Finds the first occurrence of character `c` in string `str`.
**Parameters:**
- `str` - String to search in
- `c` - Character to find (passed as int, used as char)
**Returns:**
- Pointer to first occurrence of `c`
- `NULL` if `c` is not found
**Example:**
```c
char *pos = strchr("Hello World", 'o');  // Points to first 'o' in "Hello"
```
---
### `strrchr`
```c
char *strrchr(const char *str, int c);
```
**Purpose:** Finds the last occurrence of character `c` in string `str`.
**Parameters:**
- `str` - String to search in
- `c` - Character to find
**Returns:**
- Pointer to last occurrence of `c`
- `NULL` if `c` is not found
**Example:**
```c
char *pos = strrchr("Hello World", 'o');  // Points to 'o' in "World"
```
---
### `strstr`
```c
char *strstr(const char *haystack, const char *needle);
```
**Purpose:** Finds the first occurrence of substring `needle` in string `haystack`.
**Parameters:**
- `haystack` - String to search in
- `needle` - Substring to find
**Returns:**
- Pointer to first occurrence of `needle`
- `NULL` if `needle` is not found
**Example:**
```c
char *pos = strstr("Hello World", "Wor");  // Points to "World" in string
```
---
### `strspn`
```c
size_t strspn(const char *str1, const char *str2);
```
**Purpose:** Returns the length of the initial segment of `str1` that consists entirely of characters in `str2`.
**Parameters:**
- `str1` - String to examine
- `str2` - String containing accepted characters
**Returns:** Number of characters in initial segment
**Example:**
```c
size_t len = strspn("hello123", "helo");  // Returns 5 (all of "hello")
```
---
### `strcspn`
```c
size_t strcspn(const char *str1, const char *str2);
```
**Purpose:** Returns the length of the initial segment of `str1` that consists entirely of characters NOT in `str2`.
**Parameters:**
- `str1` - String to examine
- `str2` - String containing rejected characters
**Returns:** Number of characters before any character from `str2` is found
**Example:**
```c
size_t len = strcspn("hello123", "0123456789");  // Returns 5 (stops at '1')
```
---
### `strpbrk`
```c
char *strpbrk(const char *str1, const char *str2);
```
**Purpose:** Finds the first character in `str1` that matches any character in `str2`.
**Parameters:**
- `str1` - String to search in
- `str2` - String containing characters to match
**Returns:**
- Pointer to first matching character
- `NULL` if no match found
**Example:**
```c
char *pos = strpbrk("hello world", "aeiou");  // Points to 'e' in "hello"
```

# String Tokenization Functions

### `strtok`
```c
char *strtok(char *str, const char *delim);
```
**Purpose:** Breaks string into tokens separated by delimiter characters.
**Parameters:**
- `str` - String to tokenize (first call) or `NULL` (subsequent calls)
- `delim` - String containing delimiter characters
**Returns:**
- Pointer to next token
- `NULL` when no more tokens
**Notes:**
- Modifies the original string
- Not thread-safe
- Uses internal static storage
**Example:**
```c
char str[] = "apple,banana,cherry";
char *token = strtok(str, ",");
while (token != NULL) {
    printf("%s\n", token);
    token = strtok(NULL, ",");
}
```
---
### `strtok_r`
```c
char *strtok_r(char *str, const char *delim, char **saveptr);
```
**Purpose:** Thread-safe version of `strtok`.
**Parameters:**
- `str` - String to tokenize or `NULL`
- `delim` - Delimiter characters
- `saveptr` - Pointer to save tokenization state
**Returns:** Same as `strtok`
**Notes:** Thread-safe, requires user to provide state storage.

# Memory Functions

### `memcpy`
```c
void *memcpy(void *dest, const void *src, size_t n);
```
**Purpose:** Copies `n` bytes from `src` to `dest`.
**Parameters:**
- `dest` - Destination memory
- `src` - Source memory
- `n` - Number of bytes to copy
**Returns:** Pointer to `dest`
**Notes:** Undefined behavior if memory regions overlap.
**Example:**
```c
char src[] = "Hello";
char dest[10];
memcpy(dest, src, 6);  // Copies "Hello\0"
```
---
### `memmove`
```c
void *memmove(void *dest, const void *src, size_t n);
```
**Purpose:** Copies `n` bytes from `src` to `dest`, handling overlapping memory safely.
**Parameters:**
- `dest` - Destination memory
- `src` - Source memory
- `n` - Number of bytes to copy
**Returns:** Pointer to `dest`
**Notes:** Safe for overlapping memory regions.

---
### `memcmp`
```c
int memcmp(const void *ptr1, const void *ptr2, size_t n);
```
**Purpose:** Compares first `n` bytes of memory areas.
**Parameters:**
- `ptr1` - First memory area
- `ptr2` - Second memory area
- `n` - Number of bytes to compare
**Returns:**
- `0` if memory areas are equal
- Negative if `ptr1` < `ptr2`
- Positive if `ptr1` > `ptr2`
---
### `memchr`
```c
void *memchr(const void *ptr, int value, size_t n);
```
**Purpose:** Searches for first occurrence of `value` in first `n` bytes of memory.
**Parameters:**
- `ptr` - Memory area to search
- `value` - Byte value to find
- `n` - Number of bytes to search
**Returns:**
- Pointer to first occurrence
- `NULL` if not found
---
### `memset`
```c
void *memset(void *ptr, int value, size_t n);
```
**Purpose:** Sets first `n` bytes of memory area to specified value.
**Parameters:**
- `ptr` - Memory area to set
- `value` - Value to set (converted to unsigned char)
- `n` - Number of bytes to set
**Returns:** Pointer to `ptr`
**Example:**
```c
char buffer[100];
memset(buffer, 0, sizeof(buffer));  // Clear buffer to all zeros
```

# String Length and Transformation
### `strlen`
```c
size_t strlen(const char *str);
```
**Purpose:** Returns the length of string (number of characters before null terminator).
**Parameters:**
- `str` - String to measure
**Returns:** Number of characters in string (excluding null terminator)
**Example:**
```c
size_t len = strlen("Hello");  // Returns 5
```
---
### `strxfrm`
```c
size_t strxfrm(char *dest, const char *src, size_t n);
```
**Purpose:** Transforms string according to current locale for use with `strcmp`.
**Parameters:**
- `dest` - Destination buffer
- `src` - Source string to transform
- `n` - Size of destination buffer
**Returns:** Length of transformed string
**Notes:** Used for locale-aware string comparison optimization.

# Error Handling Functions

### `strerror`
```c
char *strerror(int errnum);
```
**Purpose:** Returns a string describing the error code.
**Parameters:**
- `errnum` - Error number (typically from `errno`)
**Returns:** Pointer to error description string
**Example:**
```c
printf("Error: %s\n", strerror(errno));
```

# Best Practices and Safety Notes
1. **Buffer Overflow Prevention:**
    - Always use bounded functions (`strncpy`, `strncat`, `strncmp`) when possible
    - Ensure destination buffers are large enough
    - Consider using safer alternatives like `snprintf` for string formatting
2. **Null Termination:**
    - `strncpy` may not null-terminate; always ensure proper termination
    - Check return values from search functions for `NULL`
3. **Memory Management:**
    - Use `memmove` instead of `memcpy` when memory might overlap
    - Be careful with `strtok` as it modifies the original string
4. **Locale Considerations:**
    - Use `strcoll` and `strxfrm` for locale-aware comparisons
    - Be aware that character encoding affects comparison results
5. **Thread Safety:**
    - `strtok` is not thread-safe; use `strtok_r` in multi-threaded programs
    - `strerror` may not be thread-safe; consider `strerror_r`
